4月 13, 2019
Windows

Windows 链接匿名 SMB 共享

Windows 10 1709 后默认不允许匿名链接 SMB

解决办法是：

  1. 启用或关闭 Windows 功能

  2. 勾选 “SMB 1.0/CIFS 文件共享支持”

  3. 保存并重启电脑即可
